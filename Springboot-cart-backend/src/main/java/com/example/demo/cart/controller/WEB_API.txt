WEB API 功能
-------------------------------------------------------------------
Servlet-Path: "/products"
-------------------------------------------------------------------
GET  "/"     查詢所有商品(多筆) 範例: /products/
GET  "/{id}" 查詢指定商品(單筆) 範例: /products/1
POST "/"     新增商品        範例: /products/

-------------------------------------------------------------------
Servlet-Path: "/orders"
-------------------------------------------------------------------
GET  "/"         查詢該用戶(已登入)的所有商品(多筆) 範例: /orders/
POST "/checkout" 該用戶(已登入)進行結帳          範例: /checkout

-------------------------------------------------------------------
Servlet-Path: "/auth"
-------------------------------------------------------------------
POST "/login"      登入                範例: /auth/login
GET  "/logout"     登出                範例: /auth/logout
GET  "/isLoggedIn" 判斷目前使用者是否有登入 ? 範例: /auth/isLoggedIn

-------------------------------------------------------------------
Servlet-Path: "/favorites"
-------------------------------------------------------------------
GET    ""                     獲取用戶關注清單          範例: /favorites 
GET    "/product/{productId}" 獲取商品被那些用戶所關注的清單 範例: /favorites/product/1
POST   "/{productId}"         該用戶(已登入)加入關注商品   範例: /favorites/1
DELETE "/{productId}"         該用戶(已登入)取消關注商品   範例: /favorites/1
